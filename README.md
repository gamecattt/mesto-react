# Место

## Cайт о красивых местах в России

### Функциональность:

- Редактирование профиля
- Просмотр добавленных постов
- Создание поста
- Удаление поста
- Добавление отметки "Нравится"
- Просмотр полноразмерного фото поста

### Используемые технологии:

- React
- HTML5
- CSS3
- CSS Flexbox
- CSS Grid
- CSS Media Queries
- CSS Animations
- JavaScript
- BEM
- GIT
- DOM
- NPM
- Webpack
- OOP
